**Ч [Н]** Воспо́мни, Господи, по Своей благости чад **(или:** рабов**)** Твоих, / и всё, чем в жизни они согрешили, прости, / ведь никто не безгрешен – только Ты, // Тот, Кто может и отшедшим подать покой.

По глубочайшей премудрости Своей, единый Создатель, / Ты человеколюбиво всё устраиваешь и всем благо подаёшь, — / упокой же, Господи, души слуг **(или:** чад**)** Твоих, / ведь надежду возложи́ли они на Тебя — // Творца, и Ваятеля, и Бога нашего.

**Слава. Кондак, глас 8:**

Со святыми упокой, Христе, / души чад **(или:** рабов**)** Твоих, / где нет ни страдания, ни скорби, ни стенания, // но жизнь бесконечная.

**И ныне. Богородичен:**

Блаженной именуем тебя из рода в род, Богородица Дева, / ибо невмести́мый Христос Бог наш в тебя изволил вмести́ться. / Блаженны и мы, имеющие Засту́пницей тебя, / ибо день и ночь ты молишься за нас, / **[**`и ски́петры царств твоими мольба́ми утверждаются.`**]** / Посему мы, воспевая, возглашаем тебе: // радуйся, Благодатная, с тобою Господь!
