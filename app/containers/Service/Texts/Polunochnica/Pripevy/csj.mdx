**Ч [Н]** ~~1~~ Досто́йно есть я́ко вои́стинну, / сла́вити Тя Бо́га Сло́ва, / Его́же трепе́щут и трясу́тся Херуви́ми, / и славосло́вят Си́лы Небе́сныя, / воскре́сшаго тридне́вно из гро́ба // Христа́ Жизнода́вца, стра́хом просла́вим.

~~2~~ Воспои́м вси боголе́пно пе́сньми Боже́ственными, / Отца́, и Сы́на, и Ду́ха Боже́ственнаго, // Триипоста́сную Держа́ву, еди́но Ца́рство и Госпо́дьство.

~~3~~ Ю́же пою́т вси земноро́днии, / и славосло́вят Си́лы Небе́сныя, / от всех ве́рно поклоня́емую, // Еди́ницу по существу́, Триипоста́сную.

~~4~~ Господонача́льную Херуви́м / и Богонача́льную без сравне́ния Серафи́м, / неразде́льную Тро́ицу во Еди́нице, // су́щее Богонача́лие, Тя велича́ем.

~~5~~ Отцу́ Безнача́льному и Бо́гу, / Собезнача́льному Сло́ву со Ду́хом покланя́юся: / неразлу́чное еди́но соединя́емое Существо́, // тричи́сленную Еди́ницу пе́сньми почти́м.

~~6~~ Лучеза́рная Твоя́ мо́лния возсия́й ми, / Бо́же мой, Триипоста́сне Вседе́телю, / и дом мя покажи́ Твоея́ непристу́пныя сла́вы, // све́тел и светоно́сен и неизме́нен.

~~7~~ Его́же трепе́щут и трясу́тся Херуви́ми, / и славосло́вят А́нгельская Во́инства, / от Де́вы неизрече́нно вопло́щшагося // Христа́ Жизнода́вца, стра́хом просла́вим.
